# Documentation

This folder contains the documentation files for the Smart Energy Meter / Smart Plug project.

## Files

- `Project_Overview.pdf` - Project purpose, features, and system architecture.
- `Smart_Energy_Meter_User_Guide.pdf` - Customer/user guide.
- `App_User_Guide.pdf` - Mobile app usage guide.
- `App_Developer_Guide.pdf` - How engineers edit, run, and build the Flutter app.
- `Pin_Connection_Table.pdf` - ESP32 pin connection table.
- `Circuit_Diagram.pdf` - Clear system-level circuit/block diagram.

## Safety

This project works near AC mains voltage. Final wiring must be enclosed, insulated, protected, and checked by a qualified person.
