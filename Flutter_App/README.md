Flutter mobile app files for Smart Energy Meter.


## How a New User Installs and Uses the App

### 1. Get the Mobile App

The new user can install the Smart Energy Meter app using one of these methods:

**Method 1: APK Download Link**

1. Open the shared APK download link.
2. Download the file named `SmartEnergyMeter.apk`.
3. Tap the APK file on the phone.
4. If Android asks permission, enable **Install unknown apps**.
5. Tap **Install**.
6. Open the Smart Energy Meter app.

**Method 2: Google Play Store**

1. Open the Play Store link.
2. Tap **Install**.
3. Open the Smart Energy Meter app after installation.

---

### 2. Connect the Smart Plug to Wi-Fi

1. Power ON the smart plug.
2. Long press the Wi-Fi setup button on the device.
3. The OLED display will show the setup Wi-Fi name, for example `SmartPlug_xxxx`.
4. On the phone, go to Wi-Fi settings.
5. Connect to `SmartPlug_xxxx`.
6. Enter the setup password shown on the device.
7. Open the Smart Energy Meter app.
8. Go to **Settings → ESP32 Wi-Fi Setup**.
9. Enter the home Wi-Fi name and password.
10. Tap **Send Wi-Fi to ESP32**.
11. The smart plug will restart and connect to the user’s Wi-Fi.

---

### 3. Use the Smart Plug

After connection, the user can use the app to:

* Turn the plug ON or OFF
* View voltage, current, power, energy, and cost
* Set an auto-OFF timer
* View usage history
* Analyze energy and cost for selected time periods
* Change unit price in settings

---

### 4. Normal Use

After the first setup, the user only needs to:

1. Power ON the smart plug.
2. Open the mobile app.
3. Control and monitor the device from the app.

The setup process is needed only when connecting the smart plug to a new Wi-Fi network or a new user phone.
